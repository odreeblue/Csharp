1. 개요
- 데이터 베이스란?
  : 여러 사람이 "공유"할 목적으로 통합하여 관리하는 데이터 집합
- 데이터베이스 관리 시스템(DBMS), <data base management system>
  : MS SQL Server, Oracle, MySQL
- 용어
  * 테이블(table) : 실제 데이터를 저장하는 집합의 대표명이고 레코드 집합
  * 레코드(record) : 테이블의 각 행을 말하며 코드도 행(row)이라는 명칭으로 많이 사용
  * 필드(field) : 레코드는 다수의 항목으로 구분되며 이 항목을 필드라고 함
  * 컬럼(column) : 컬럼은 필드(열)을 구분하기 위한 대표명

2. ADO.NET
- C# 과 .Net이라는 것이 들어오면서 데이터베이스를 원활하게 OS와 상관없이
  ADO.net이 설치되어있는 곳에는 C#프로그램이 동작할 수 있게끔 해주는 모듈
- 과거에는 odbc라는 것이 os에 탑재가 되어있었음
- 이제는 ADO.NET <-> .NET과 연관되면서 데이터베이스와 관련된 부분을 담당하게 됨
